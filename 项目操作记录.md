## 5. 常见问题记录

- `git push origin main` 失败：因为该仓库分支是 `gh-pages`，不是 `main`。
- GitHub 推送偶发失败（443 连接重置）：属于网络波动，重试可恢复。
- 页面白屏：通常是 `index.html` 与 `assets/index-*.js/css` 不匹配或路径错误。
- 安全提醒：`sb_secret_` 是服务端私钥，不能用于前端，必须立即重置后弃用。
