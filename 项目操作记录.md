## 2. 标准部署命令（直接复制）

```powershell
cd "d:\重要文件\Desktop\tutuai网站"

# 全部部署（推荐）
npm run deploy:all

# 或分步：
npm run deploy:netlify   # Netlify 站点 + 云端 API
npm run deploy:pages     # GitHub Pages + Gitee
npm run deploy:source    # 推送源码到 GitHub
```

## 4. 同步问题排查

1. **Ctrl + F5** 强刷，避免旧 JS 缓存
2. 看右上角状态：「云端已同步 · Netlify 云端」为正常
3. 失败时点 **「重试」**
4. 验证 API：  
   `https://beautiful-basbousa-c7556d.netlify.app/.netlify/functions/app-state?meta=1`
5. GitHub push 443 重置 → 稍后重试
